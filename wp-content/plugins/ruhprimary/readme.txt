A custom Elementor addon developed for Ruh Primary, providing purpose-built widgets for curriculum, school life, afterschool programs, and more.

== Description ==

**Ruhprimary Addon** extends Elementor with custom educational components designed specifically for Ruh Primary.  
With this plugin, you can easily drag-and-drop engaging sections like:

- Home Page Layout
- Parent Community Overview
- Curriculum Details
- About Us Page
- School Life (Life@RUH)
- Sports & Enrichment Programs
- Contact Us Page
- Quick Links for Students & Parents

All widgets are grouped under the “FOS Widgets” category in the Elementor editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ruhprimary-addon` directory, or install the plugin through the WordPress dashboard.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Make sure Elementor is installed and active.
4. Open any page using Elementor and look under the **FOS Widgets** category to start using the custom blocks.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =  
No, it works with the free version of Elementor.

= Can I use this plugin outside Ruh Primary? =  
It’s tailored for Ruh Primary’s content structure, but you can adapt it for other schools with similar needs.
